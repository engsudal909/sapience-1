# Build a Trading Agent

This guide shows your agent can request odds for a wager it would like to place on a forecast.

## Start auctions and receive bids

```ts
const ws = new WebSocket('ws://localhost:3001/auction');

ws.onopen = () => {
  ws.send(JSON.stringify({
    type: 'auction.start',
    payload: {
      taker: '0xTaker...',
      wager: '1000000000000000000',
      predictions: [{
        verifierContract: '0xVerifier...',
        resolverContract: '0xResolver...',
        predictedOutcomes: '0xabc...',
      }],
      takerNonce: 1,
      chainId: 42161,
      marketContract: '0xPredictionMarket...',
    },
  }));
};

ws.onmessage = (ev) => {
  const msg = JSON.parse(String(ev.data));
  if (msg.type === 'auction.ack') {
    console.log('auctionId =', msg.payload.auctionId);
  }
  if (msg.type === 'auction.bids') {
    const best = pickBest(msg.payload.bids);
    console.log('best bid =', best);
  }
};

function pickBest(bids) {
  const now = Math.floor(Date.now()/1000);
  const valid = bids.filter(b => b.makerDeadline > now);
  valid.sort((a, b) => BigInt(b.makerWager) - BigInt(a.makerWager) > 0n ? 1 : -1);
  return valid[0];
}
```

Takers initiate auctions by sending `auction.start`. The relayer broadcasts `auction.started` with parameters including `auctionId`, `taker`, `wager`, `predictions[]`, `chainId`, and `marketContract`.

## Accept a Bid

After selecting a bid, call `PredictionMarket.mint` using the current auction parameters and the selected bid fields.

```ts
import { writeContract } from 'viem/actions';
import { erc20Abi } from 'viem';
import { abi as PredictionMarketAbi } from './PredictionMarket.abi';

// Approve taker collateral for PredictionMarket to pull (the auction starter)
await writeContract({
  address: '0xCollateralToken...',
  abi: erc20Abi,
  functionName: 'approve',
  args: ['0xPredictionMarket...', '1000000000000000000'], // takerCollateral (auction starter)
});

const bestBid = /* from auction.bids */
  {
    maker: '0xMaker...',
    makerWager: '500000000000000000',
    makerDeadline: Math.floor(Date.now()/1000) + 60,
    makerSignature: '0x' + '11'.repeat(32) + '22'.repeat(32),
    makerNonce: 1,
  };

const mintReq = {
  encodedPredictedOutcomes: '0xabc...',
  resolver: '0xResolver...',
  makerCollateral: bestBid.makerWager,
  takerCollateral: '1000000000000000000',
  maker: bestBid.maker,
  taker: '0xTaker...',
  takerSignature: bestBid.makerSignature,
  takerDeadline: String(bestBid.makerDeadline),
  refCode: '0x' + '00'.repeat(32),
  makerNonce: String(bestBid.makerNonce),
};

await writeContract({
  address: '0xPredictionMarket...',
  abi: PredictionMarketAbi,
  functionName: 'mint',
  args: [mintReq],
});
```

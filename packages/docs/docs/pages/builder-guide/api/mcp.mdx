# Model Context Protocol (MCP)

## What is MCP?

MCP is an open standard for connecting AI applications to external data sources and tools. It lets AI assistants like Claude or ChatGPT access live data and perform actions on your behalf.

Learn more at **[modelcontextprotocol.io](https://modelcontextprotocol.io/)**

## Endpoint

```
https://api.sapience.xyz/mcp
```

Connect any [MCP client](https://modelcontextprotocol.io/clients) to query open questions, explore existing positions, generate calldata for blockchain transactions, and more.

## Quick Setup

Add to Claude Desktop or Cursor via `mcp-remote`:

```json
{
  "mcpServers": {
    "sapience": {
      "command": "npx",
      "args": ["mcp-remote", "https://api.sapience.xyz/mcp"]
    }
  }
}
```

No authentication required. Please be considerate; rate limiting may apply.
